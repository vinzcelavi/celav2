## Run locally

```
pnpm install
pnpm run dev
```

## Tech Stack

- [Vite](https://vitejs.dev/) - Framework.
- [Tailwind CSS](https://tailwindcss.com/) - Styling.
- [Framer Motion](https://www.framer.com/motion/) - Animation.

## Run Locally

```bash
$ git clone https://github.com/vinzcelavi/celav2-fr.git
$ cd celav2-fr
$ pnpm install
$ pnpm dev
```
